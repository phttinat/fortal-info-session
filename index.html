พูดบอก AI ว่าอยากได้เกมแบบไหน แล้วมันสร้างให้ ทดสอบบนซิมูเลเตอร์ที่บ้าน จากนั้นกดปล่อยลงฮาร์ดแวร์ Arena จริง

เปลี่ยนคนเล่นให้เป็นคนสร้าง เปิดทางให้ทุกคนทำเกมได้ บนโครงสร้างระดับมืออาชีพของ Fortal

บทพูด (~1 นาที) และนี่คือที่เรากำลังมุ่งไป FortalOS. จินตนาการว่าคุณแค่พูดกับ AI ว่าอยากได้เกมแบบไหน มันสร้างให้ ลองเล่นบนซิมที่บ้าน แล้วกดปล่อยลงสนามจริงได้เลย. เราอยากเปลี่ยนคนเล่นให้เป็นคนสร้าง. นี่คือวิสัยทัศน์ที่ทีมเรากำลังบุกเบิก ถ้าคุณสนใจ AI บวกเกม นี่คือของจริง.

บทพูด (~1.5 ถึง 2 นาที) สำคัญกับนักศึกษา ทีนี้ส่วนที่เกี่ยวกับพวกคุณโดยตรง. มาทำงานที่นี่ คุณจะได้แตะของพวกนี้ของจริง ไม่ใช่โปรเจกต์เล่นๆ. Unity และ game dev เต็มรูปแบบ, Mixed Reality บน Meta Quest, ระบบ tracking UWB, งานฮาร์ดแวร์ LED กับ haptic, AI สร้างเกม, และงานครีเอทีฟทั้งอาร์ต บท เสียง. ไม่ว่าคุณจะสายโค้ด สายอาร์ต หรือสายฮาร์ดแวร์ เรามีที่ให้.

Test & Quality

QA / Tester

ทดสอบเกม โปรแกรม และฮาร์ดแวร์ (manual & exploratory) เขียน bug report ที่ทำซ้ำได้ และออกแบบ test case ให้ครอบคลุม

คุณสมบัติ
  • เข้าใจพื้นฐานการทดสอบซอฟต์แวร์ รู้จัก test case และ bug report
  • ช่างสังเกต ละเอียด อธิบายปัญหาให้คนอื่นทำตามซ้ำได้
  • สื่อสารกับทีมพัฒนารู้เรื่อง ทำงานเป็นทีมได้

ถ้ามีจะดีมาก: automated test, Git หรือเครื่องมือ track bug

AI & Automation

AI Engineering

ศึกษา ค้นคว้า และพัฒนาเครื่องมือด้วย AI ประยุกต์ใช้ AI ช่วยงานทดสอบและกระบวนการพัฒนาเกม

คุณสมบัติ
  • มีพื้นฐานการเขียนโปรแกรม (เช่น Python, C# ภาษาใดก็ได้)
  • เข้าใจพื้นฐาน Machine Learning / LLM ว่าทำงานยังไง
  • ใช้ Claude, ChatGPT, Gemini เป็น เขียน prompt ให้ได้ผลลัพธ์ดี

ถ้ามีจะดีมาก: เคยทำโปรเจกต์กับ AI API หรืองาน automation
